# Input输入类型：range

range 类型用于应该包含一定范围内数字值的输入域。

range 类型显示为滑动条。

还能够设定对所接受的数字的限定：

例：

```javascript
<!DOCTYPE HTML>
<html>
<body>

<form action="/example/html5/demo_form.asp" method="get">
Points: <input type="range" name="points" min="1" max="10" />
<input type="submit" />
</form>

</body>
</html>
```
